# 推荐任务经典论文列表

以下论文按当前赛题数据的适用程度排序。当前数据具有测试用户历史极短、冷启动比例高、重复目标明显、训练与测试用户不重合等特点，因此应优先阅读重复推荐、商品协同、序列转移和学习排序方向。

## 一、优先阅读

### 1. RepeatNet: A Repeat Aware Neural Recommendation Machine for Session-Based Recommendation

- 作者：Pengjie Ren 等
- 会议：AAAI 2019
- 论文：[AAAI 官方页面](https://ojs.aaai.org/index.php/AAAI/article/view/4408)
- 关键词：重复推荐、探索推荐、会话推荐
- 与本赛题的关系：训练集中大量目标商品已经在用户历史中出现，适合借鉴 RepeatNet 的 `repeat / explore` 双分支思想。

### 2. Item-Based Collaborative Filtering Recommendation Algorithms

- 作者：Badrul Sarwar、George Karypis、Joseph Konstan、John Riedl
- 会议：WWW 2001
- 论文：[原论文页面](https://ra.ethz.ch/cdstore/www10/papers/519/sdm2.html)
- DOI：[10.1145/371920.372071](https://doi.org/10.1145/371920.372071)
- 关键词：ItemCF、商品相似度、协同过滤
- 与本赛题的关系：可以从用户历史商品召回相似商品，适合作为轻量候选生成方法。

### 3. Embarrassingly Shallow Autoencoders for Sparse Data

- 作者：Harald Steck
- 会议：WWW 2019
- 论文：[arXiv](https://arxiv.org/abs/1905.03375)
- DOI：[10.1145/3308558.3313710](https://doi.org/10.1145/3308558.3313710)
- 关键词：EASE、稀疏隐式反馈、线性推荐
- 与本赛题的关系：适合从用户—商品交互矩阵学习商品关联；A榜商品规模较小时可作为强基线。

### 4. Factorizing Personalized Markov Chains for Next-Basket Recommendation

- 作者：Steffen Rendle、Christoph Freudenthaler、Lars Schmidt-Thieme
- 会议：WWW 2010
- 论文：[原论文 PDF](https://archives.iw3c2.org/www2010/pub/pdfs/RendleFreudenthaler2010-FPMC.pdf)
- DOI：[10.1145/1772690.1772773](https://doi.org/10.1145/1772690.1772773)
- 关键词：FPMC、矩阵分解、Markov 转移、序列推荐
- 与本赛题的关系：适合学习“长期偏好 + 最近商品转移”。由于测试用户均为新用户，应重点借鉴商品转移部分，谨慎使用用户 ID 嵌入。

### 5. From RankNet to LambdaRank to LambdaMART: An Overview

- 作者：Christopher J. C. Burges
- 机构：Microsoft Research，2010
- 论文：[Microsoft Research 官方页面](https://www.microsoft.com/en-us/research/publication/from-ranknet-to-lambdarank-to-lambdamart-an-overview/)
- 关键词：Learning to Rank、LambdaRank、LambdaMART、NDCG
- 与本赛题的关系：可将热度、Repeat、ItemCF、EASE、Markov、用户特征和商品特征统一交给 LightGBM LambdaRank 等排序器融合。

### 6. Factorization Machines

- 作者：Steffen Rendle
- 会议：ICDM 2010
- 论文：[DOI 页面](https://doi.org/10.1109/ICDM.2010.127)
- 关键词：Factorization Machine、稀疏特征交叉、推荐
- 与本赛题的关系：适合学习匿名用户类别特征与商品类别特征之间的低阶交互，尤其适用于无历史用户。

### 7. BPR: Bayesian Personalized Ranking from Implicit Feedback

- 作者：Steffen Rendle、Christoph Freudenthaler、Zeno Gantner、Lars Schmidt-Thieme
- 会议：UAI 2009
- 论文：[arXiv](https://arxiv.org/abs/1205.2618)
- 关键词：隐式反馈、两两排序、负采样
- 与本赛题的关系：适合作为“正商品得分高于负商品”的排序损失，但传统用户 ID 矩阵分解无法直接泛化到未见测试用户。

## 二、序列模型扩展

### 8. Session-Based Recommendations with Recurrent Neural Networks

- 作者：Balázs Hidasi、Alexandros Karatzoglou、Linas Baltrunas、Domonkos Tikk
- 会议：ICLR 2016
- 论文：[arXiv](https://arxiv.org/abs/1511.06939)
- 关键词：GRU4Rec、短会话、序列推荐、Ranking Loss
- 与本赛题的关系：可以把用户历史看作短会话，但测试序列过短时需要与热门、Repeat等方法融合。

### 9. Self-Attentive Sequential Recommendation

- 作者：Wang-Cheng Kang、Julian McAuley
- 会议：ICDM 2018
- 论文：[arXiv](https://arxiv.org/abs/1808.09781)
- 关键词：SASRec、自注意力、序列推荐
- 与本赛题的关系：可学习历史中不同商品的重要性，适合作为序列模型候选；不建议在冷启动用户上单独使用。

## 三、图推荐扩展

### 10. LightGCN: Simplifying and Powering Graph Convolution Network for Recommendation

- 作者：Xiangnan He 等
- 会议：SIGIR 2020
- 论文：[arXiv](https://arxiv.org/abs/2002.02126)
- DOI：[10.1145/3397271.3401063](https://doi.org/10.1145/3397271.3401063)
- 关键词：LightGCN、用户—商品二部图、协同过滤
- 与本赛题的关系：对应赛题的图链接预测表述，但标准 LightGCN 依赖用户 ID 嵌入；测试用户未在训练中出现，因此需要改造成基于历史邻居聚合的归纳式版本。

## 建议阅读顺序

1. RepeatNet
2. ItemCF
3. EASE
4. FPMC
5. LambdaMART
6. Factorization Machines
7. BPR
8. GRU4Rec 与 SASRec
9. LightGCN
